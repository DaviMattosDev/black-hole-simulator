# Black Hole Lab 3D — Davi Mattos

Versão focada em uma experiência interativa de buraco negro 3D, horizonte de eventos e deformação gravitacional visual.

## Principais recursos

- Buraco negro 3D com disco de acreção e partículas orbitando.
- Malha de espaço-tempo deformada e geodésicas visuais de luz.
- Modo espectador sem HUD.
- Entrada no horizonte em modo imersivo: todos os parâmetros principais vão para o máximo.
- Legenda única em estilo Star Wars, mais lenta e fluida.
- Efeito final de espaguetificação: letras giram em espiral/caracol para dentro.
- Interface em Português e Inglês.
- Modo claro e modo escuro.
- Seção com cálculos e teorias usadas.

## Rodar localmente

```bash
npm config set registry https://registry.npmjs.org/
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Atalhos

- `S`: alterna modo espectador.
- `L`: alterna idioma PT/EN.
- `T`: alterna tema claro/escuro.
- `ESC`: cancela a entrada ou sai do modo espectador.

## Observação

Esta experiência é educativa e artística. Os cálculos principais usam constantes físicas reais e aproximações conhecidas, mas os efeitos visuais de lente, geodésicas e espaguetificação são representações visuais simplificadas.
